📲 홈 화면에 앱 추가하는 방법

이 앱은 크롬에서 실행한 후 왼 쪽 상단에 있는 ☰ 버튼을 누르면 맨 하단에 홈 화면에 설치 메뉴가 보입니다. 이를 실행하면 자동 설치가 가능합니다.

QR코드로 링크주소를 받으면 바로 크롬으로 열리게 됨으로 메뉴에서 홈 화면에 설치 메뉴를 실행하면 설치가 됩니다.

스마트폰 기본 설정에 따라 다른 브라우저로 [기도서]를 열었으면 아래의 방법으로 크롬브라우저로 다시 열어야 앱으로 설치가 됩니다.

삼성 브라우저나 카톡 브라우저로 [기도서] 앱을 열면 홈 화면에 추가 메뉴가 보이지 않습니다. 하지만, 다음과 같은 방법들로 우회하여 크롬 브라우저로 열 수 있습니다:

1. 삼성 브라우저

  1. 링크 복사 후 붙여넣기 방법: 삼성 브라우저 상단 주소창에 보이는 웹사이트의 URL 주소를 길게 눌러 복사한 후, 홈 버튼으로 빠져나와서 크롬 브라우저 앱을 실행하여 주소창에 붙여넣어 다시 [기도서]를 열 수 있습니다.
  2. 공유 기능 활용 방법: 삼성 브라우저에서 오른 쪽 하단에 있는 ☰ 버튼을 눌러서 공유 기능을 사용하여 크롬 브라우저를 실행하여 [기도서]를 다시 열 수 있습니다.

    크롬으로 실행되면 [기도서] 앱의 왼쪽 상단에 ☰ 버튼을 눌러서 홈 화면에 추가 메뉴를 실행하면 앱이 설치되고 아이콘도 보입니다. 설치가 되면 홈 화면에 설치 메뉴는 더 이상 보이지 않습니다.

2. 카카오톡 브라우저

  1. 오른쪽 하단 ⋮ 더보기 메뉴를 누르세요.
  2. 링크복사 메뉴로 링크 주소를 복사한 후에 홈 버튼으로 나와서 다시 크롬을 실행하여 주소창 링크 주소를 복사한 후 크롬으로 실행합니다.

    크롬으로 실행되면 [기도서] 앱의 왼쪽 상단에 ☰ 버튼을 눌러서 홈 화면에 추가 메뉴를 실행하면 앱이 설치되고 아이콘도 보입니다.

💡 홈 화면에 추가된 아이콘은 일반 앱처럼 오프 라인에서도 바로 실행할 수 있습니다.
