📲 홈 화면에 앱 추가하는 방법

이 앱은 삼성, 크롬 브라우저에서 실행한 후 왼 쪽 상단에 있는 ☰ 버튼을 누르면 맨 상단에 홈 화면에 설치 메뉴가 보입니다. 이를 실행하면 자동 설치가 됩니다. 설치가 다 되면 앱 아이콘이 보입니다. 이 아이콘을 홈 화면으로 복사해줍니다.

아이폰에서는 PWA프로그램 지원을 안하므로 웹으로만 연결됩니다. 이때는 브라우저 하단 메뉴에서 공유 버튼 클릭해서 사이트 바로가기 를 만들어 주면 앱처럼 사용할 수 있습니다.

카톡에서 링크를 클릭해서 [기도서] 화면이 열렸으면 이때도 홈 화면에 추가 메뉴가 보이지 않습니다. 이때는 화면 아래 ⋮ 표시 버튼을 눌러 다른 브라우저로 열기를 하면 설치 메뉴가 보입니다.

간혹 오타 수정되거나 업데이트되는 경우 오류 화면이 뜨는 경우가 있으므로 이때는 ☰ 버튼누르고 버전 업데이트메뉴를 실행해 주세요.

💡 홈 화면에 추가된 아이콘은 일반 앱처럼 오프 라인에서도 바로 실행할 수 있습니다.
